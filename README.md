# **`touch`** by 4lo8ecK
Это просто аналог `GNU touch` для использования в консоли на Windows. Создан для собственного использования.

### Возможности
- [x] создание пустого файла
- [ ] запись в файл при его создании с помощью `>`
- [ ] изменение времени последнего редактирования файла

### Компиляция и установка
1. Скомпилируйте любым удобным способом.
    Например, `gcc touch.c -o touch`. В итоге получаем `touch.exe`
2. Переместите полученный `touch.exe` в любую удобную директорию.
3. Добавьте путь к директории с бинарником в `Path`:
    1. Либо через системный редактор переменных окружения в Windows 
    2. Либо командой в `cmd`, `power shell` с **правами администратора** следующую команду `setx PATH %PATH%; <путь к директории с бинарником>`
